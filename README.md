# ios-uikit-uiactivityindicatorview-demo

## 概要
歯車が回転したり、止まったすことで、タスクが進行中であることを示すクラス

### 継承元
UIView

## 実装手順
1.ストーリーボードから、UIButtonのアクションを２つとUIActivityIndicatorViewを追加</br>
2.上記をViewControllerと紐付け</br>
3.メソッド実装</br>
